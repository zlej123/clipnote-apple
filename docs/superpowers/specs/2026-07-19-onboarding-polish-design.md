# clipnote-apple v1.2: 온보딩 폴리시 설계

2026-07-19 사용자 피드백 3건(설계 승인됨). 첫 Gemini 키 입력 경험 개선 — UI 2~3파일 수정, 단일 태스크.

## 요구 (피드백 원문 기준)

1. **키 미설정 배너 시선 유도**: 홈의 주황 배너가 깜빡이며 시선을 끈다. `key.fill` 아이콘 `.symbolEffect(.pulse, options: .repeating)` + 배너 배경 주황 틴트 1.2초 페이드 반복. **접근성**: `accessibilityReduceMotion` 시 애니메이션 없이 정적 강조 배경만. 키 저장 시 배너 소멸(기존 동작).
2. **AI Studio 링크 클릭 유도**: 설정의 키 필드가 **비어 있을 때만** 링크를 프로미넌트 버튼(`arrow.up.right` 라벨)으로 승격 + 옆에 "← 여기를 눌러 1분 발급" 주황 캡션. 키가 있으면 기존 조용한 링크 유지.
3. **발급 절차 인앱 안내**: 같은 조건(키 비어 있을 때)에 `DisclosureGroup("키 발급 방법 보기")` — 텍스트 4단계(①구글 계정으로 aistudio.google.com/apikey 접속 ②"API 키 만들기(Create API key)" 클릭 ③생성된 키 복사(AIza로 시작) ④위 칸에 붙여넣고 '키 저장' — 무료·카드 불필요). 이미지 예시는 범위 밖(구글 UI 개편 시 낡음 — 사용자 승인).
